Study the Syncronization phenomenon for coupled maps in (1+1)-lattice $x(t+1) = (1 - \gamma) f(x(t)) + \gamma f(y(t))$, $y(t+1) = \gamma f(x(t)) + (1 - \gamma) f(y(t))$, where $f$ is a specific map. Specifically, the Bernoulli map $f(x) = 2x \ \text{mod} \ 1$ and Tend map ($f(x) = ax$ if $x\in[0,1/2]$ and $f(x) = a(1-x)$ if $x \in (1/2,1]$) are used.

In [BernoulliMap.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/1a8da499ec10116ffc2fd40cbcd09daaf12c74c2/Synchronization%20of%20coupled%20map/BernoulliMap.m) and [SkewTendMap.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/1a8da499ec10116ffc2fd40cbcd09daaf12c74c2/Synchronization%20of%20coupled%20map/SkewTendMap.m) the synchronization mechanism is studied fixing the parameter $\gamma$; while in [BernoulliMap2.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/1a8da499ec10116ffc2fd40cbcd09daaf12c74c2/Synchronization%20of%20coupled%20map/BernoulliMap2.m) and [SkewTendMap2.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/1a8da499ec10116ffc2fd40cbcd09daaf12c74c2/Synchronization%20of%20coupled%20map/SkewTendMap2.m), $\gamma$ varies in a range, allowing to address when the synchronization is realized or not.
