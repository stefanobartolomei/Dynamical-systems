The folder contains a brief study of the Logistic map: $x_{n+1} = r x_n (1 - x_n)$, where $r \in \mathbb{R}^+$.

The code [LogisticMap.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/e335b3a96f989ae40e47af6f7873fbff0b79fe5a/Logistic%20Map/LogisticMap.m) simulates the dynamic for different values of $r$. In [LyapunovExp.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/e335b3a96f989ae40e47af6f7873fbff0b79fe5a/Logistic%20Map/LyapunovExp.m) it's studied how the Lyapunov exponent depends on the parameter of the map $\lambda(r)$, which shows for what values of $r$ the evolution is chaotic ($\lambda>0$); while [Bifurcationdiagram.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/82341e51a0c3c46d2b8a2e63465ca06e0066e606/Logistic%20Map/Bifurcationdiagram.m) plots the bifurcation diagram varying $a$ parameter.
