The folder contains an analysis of the Henon Map: $x_{n+1} = 1 + y_n - a x_n^2$, $y_{n+1} = b x_n$ where $a$ is a free parameter and $b \in (-1,1)$.

In [Henon Map.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/d4c102d3209af5f9a23b2e92f1c147de9f137fb2/Henon%20Map/HenonMap.m) is simulated the dynamic for a specific value of the parameters ($a = 1.4$ and $b = 0.3$), chosen such that the evolution is chaotic. The characterization of the chaoticity nature of the map for those parameters' values is related with the positivity of one Lyapunov exponent $\lambda_1 >0$ so in [LyapunovExp.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/d4c102d3209af5f9a23b2e92f1c147de9f137fb2/Henon%20Map/LyapunovExp.m) both two Lyapunov exponents are estimated. Moreover, [CorrelationDimGP.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/d4c102d3209af5f9a23b2e92f1c147de9f137fb2/Henon%20Map/CorrelationDimGP.m) estimates the Correlation dimension $D(2)$, which bounds below the Fractal dimension $D_F$ of the Henon Map, whose dynamic for those values, being chaotic, draws a fractal structure. Also the Correlation function is calculated in [CorrelationF.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/9e7e9e3e01bfd22543a462395cb33b463a9f186e/Henon%20Map/CorrelationF.m).

Instead, in [LLE.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/9e7e9e3e01bfd22543a462395cb33b463a9f186e/Henon%20Map/LLE.m) and [Bifurcationdiagram.m](https://github.com/stefanobartolomei/Dynamical-systems/blob/9e7e9e3e01bfd22543a462395cb33b463a9f186e/Henon%20Map/Bifurcationdiagram.m) the maximum Lyapunov exponent and the bifurcation diagram are calculated fixing $b = 0.3$ and varying $a$, respectively. From those codes emerges for which values of $a$ the Henon map is chaotic.
