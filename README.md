The folder contains a basic analysis for the main examples of Dynamical systems (discrete and continuos):
1. In [Henon Map](https://github.com/stefanobartolomei/Dynamical-systems/tree/30fd3a662d751100b4634e8574f1ee21de1f66d8/Henon%20Map) the study of the chaotic nature of the Henon map $x_{n+1} = 1 + y_n -a x_n^2$, $y_{n+1} = b x_n$ is performed.
